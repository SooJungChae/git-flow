# git-flow

커밋별로 각 상황별로 구현해보면서 git-flow 방식을 도입해보려고 한다. 

**기대 효과**
- 순서대로 상황을 재현해보며 전체적인 flow 를 이해할 수 있다.
- 그래프를 깔끔하게 유지하는 방법을 이해한다.

**참고사이트**
- https://nvie.com/posts/a-successful-git-branching-model/

--- 

# 구현 순서

## master

## develop

## feature
- 개발완료

## release
- 개발완료
- 태그달기

## hotfix
- 버그픽스
- 버전 붙이기
